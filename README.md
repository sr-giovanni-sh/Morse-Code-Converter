# Morse Code Converter (REST API)

Инструмент для автоматической конвертации текста в азбуку Морзе и обратно через HTTP-интерфейс.

## 🚀 Возможности
- **Auto-detection**: Сервис сам определяет тип входных данных (текст или Морзе).
- **File Upload**: Загрузка через HTML-форму (`multipart/form-data`).
- **Safety**: Настроены таймауты сервера (Read/Write/Idle) для защиты от атак.

## 🛠 Запуск
```bash
go run cmd/main.go
```
Сервер запустится на [http://localhost:8080](http://localhost:8080)

## 📂 Структура
- `cmd/` — точка входа.
- `internal/` — сервер, хендлеры и логика.
- `pkg/morse/` — библиотека конвертации.
- `index.html` — интерфейс формы.
